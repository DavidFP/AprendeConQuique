=========================================================================
RECOMENDACIONES PARA LA EJECUCIÓN
=========================================================================

Para ejecutar esta aplicación basta con tener la carpeta "lib" en el mismo 
directorio en que se esté ejecutando Mates_V1.0.jar

Para ejecutarlo en entornos de escritorio basta con hacer doble click en el
icono del fichero .jar

Para ejecutarlo en consola podemos usar el siguiente comando:

java -jar "Mates_v1.0.jar"
